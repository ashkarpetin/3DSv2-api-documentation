5. Authentication Examples¶

The 3DSv2 spec allows for a lot of flexibility. A complication of the interface is that, the specification marks elements as required while card schemes have their own rules.

On this page we will provide JSON examples with the minimum key set required by all card schemes with dummy values.

To get more information on individual fields, consult the reference.
